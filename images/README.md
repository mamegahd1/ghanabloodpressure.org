# /images

Single home for all images used on the GHSI website (ghanabloodpressure.org).

## Naming convention

Use lowercase **kebab-case**, descriptive names. Examples:

- `screening-event-accra.jpg`
- `team-prince-nuamah.jpg`
- `pilot-curriculum-cover.webp`

Avoid spaces, capital letters, and generic names like `img1.jpg`.

## Size guidance

- Maximum width: **~1600px** on the long edge.
- Target file size: **under ~400 KB**.
- Format: optimized **JPG** for photographs, **WebP** where broadly supported, **PNG** only for logos or diagrams that need transparency.
